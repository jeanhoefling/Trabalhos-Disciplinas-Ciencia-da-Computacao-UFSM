# 📚 Trabalhos da Faculdade

Este repositório reúne todos os trabalhos, exercícios e projetos desenvolvidos ao longo das disciplinas do curso de graduação.

## 📌 Objetivo

Organizar e centralizar os trabalhos acadêmicos, facilitando o acompanhamento da evolução nas disciplinas e servindo como material de estudo e consulta futura.

## 🗂️ Estrutura

Os arquivos estão organizados por disciplina.

## 📖 Como usar

Basta navegar pelas pastas e abrir os arquivos correspondentes a cada disciplina ou atividade.

Se algum projeto exigir compilação ou execução específica, instruções estarão dentro da própria pasta.

## ⚠️ Observações

- Este repositório tem fins **educacionais**.
- Alguns códigos podem estar em desenvolvimento ou não totalmente otimizados.

## 📅 Organização futura

Pretendo manter este repositório atualizado conforme novas disciplinas e projetos forem sendo adicionados.
